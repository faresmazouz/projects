# Infrastructure mise en place

J'ai mis en place une infrastructure complète composée de :

- un **Windows 10** ;
- une **Debian 13** ;
- un **pare-feu pfSense**.

Dans cette infrastructure, j'ai également configuré :

- un **Active Directory** ;
- un serveur **DNS** ;
- un serveur **DHCP** ;
